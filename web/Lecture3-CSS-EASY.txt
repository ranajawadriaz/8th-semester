ğŸ“Œ SECTION 2: CSS SELECTORS - HOW TO TARGET ELEMENTS
================================================================================

ğŸ”¹ WHAT ARE SELECTORS?
   - Selectors CHOOSE which elements to style
   - Like pointing at something and saying "make THAT one red"

--------------------------------------------------------------------------------
ğŸ”¹ SELECTOR TYPE 1: ELEMENT SELECTOR
--------------------------------------------------------------------------------

   Targets ALL elements of that type:

   /* This makes ALL paragraphs red and centered */
   p {
     text-align: center;
     color: red;
   }

   EXAMPLE IN HTML:
   <p>I'm red and centered!</p>
   <p>Me too!</p>
   <p>And me!</p>
   <!-- ALL p tags get the style -->

--------------------------------------------------------------------------------
ğŸ”¹ SELECTOR TYPE 2: ID SELECTOR (uses #)
--------------------------------------------------------------------------------

   Targets ONE SPECIFIC element by its ID:
   âš ï¸ IDs should be UNIQUE (only one element with that ID!)

   /* The # means "look for ID" */
   #para1 {
     text-align: center;
     color: red;
   }

   EXAMPLE IN HTML:
   <p id="para1">I'm the special one!</p>  <!-- Only this gets styled -->
   <p>I'm normal</p>

--------------------------------------------------------------------------------
ğŸ”¹ SELECTOR TYPE 3: CLASS SELECTOR (uses .)
--------------------------------------------------------------------------------

   Targets ALL elements with that class:
   âœ… Classes can be reused on multiple elements!

   /* The . means "look for class" */
   .highlight {
     background-color: yellow;
   }

   EXAMPLE IN HTML:
   <p class="highlight">I'm highlighted!</p>    <!-- Gets styled -->
   <p>I'm not</p>
   <p class="highlight">Me too!</p>             <!-- Gets styled -->
   <h2 class="highlight">Even headings!</h2>    <!-- Gets styled -->

   â­ ONE ELEMENT CAN HAVE MULTIPLE CLASSES:
   <p class="center large">I have TWO classes!</p>
   
   .center { text-align: center; }
   .large { font-size: 24px; }
   <!-- This paragraph gets BOTH styles! -->

--------------------------------------------------------------------------------
ğŸ”¹ SELECTOR TYPE 4: UNIVERSAL SELECTOR (uses *)
--------------------------------------------------------------------------------

   Targets EVERYTHING on the page:

   /* * means "select ALL elements" */
   * {
     text-align: center;
     color: blue;
   }

   <!-- Every single element becomes centered and blue! -->

--------------------------------------------------------------------------------
ğŸ”¹ SELECTOR TYPE 5: GROUPING SELECTOR (uses ,)
--------------------------------------------------------------------------------

   Apply same style to MULTIPLE selectors at once:

   /* Instead of writing same style 3 times... */
   h1, h2, p {
     text-align: center;
     color: red;
   }

   <!-- h1, h2, AND p all get these styles -->

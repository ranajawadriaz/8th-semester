Q1: What is the difference between HTTP and HTTPS?
A1: HTTP sends data in plain text (not secure), HTTPS encrypts data (secure).
    HTTPS uses SSL/TLS encryption to protect sensitive information.

Q2: What is the difference between Web App and Desktop App?
A2: Web App runs in BROWSER, accessed via internet, no installation needed.
    Desktop App is INSTALLED on computer, works offline, stored locally.

Q3: What is the meaning of responsive design?
A3: Design approach where website ADAPTS to different screen sizes and devices
    automatically. Same website works on phone, tablet, and desktop.

Q4: Can you name an open source framework for building responsive websites?
A4: Bootstrap, Tailwind CSS, Foundation (Bootstrap is most popular)

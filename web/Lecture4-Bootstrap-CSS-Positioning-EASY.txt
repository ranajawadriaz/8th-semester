ğŸ”¹ POSITION: STICKY
--------------------------------------------------------------------------------

   - Behaves like RELATIVE until scroll reaches a threshold

ğŸ”¹ WHAT IS MOBILE-FIRST?
   - Design for SMALL screens first
   - Then add styles for LARGER screens
   - Bootstrap is built mobile-first!

ğŸ”¹ HOW IT WORKS:

   /* Base styles (for mobile) */
   .element {
     font-size: 14px;
     padding: 10px;
   }

   /* Then add for larger screens */
   @media (min-width: 768px) {
     .element {
       font-size: 16px;
       padding: 20px;
     }
   }

   In Bootstrap, col-sm-12 affects SMALL AND UP (mobile-first)
